# 抽象工厂方法代码

最新更新：`= dateformat(date(today), "yyyy-MM-dd")`

**语言**：Java
**一句话描述**: 抽象工厂相较于工厂方法，可以满足不同产品等级的需求，在抽象工厂中声明一组，制造不同产品种类（不同产品等级）的方法。

## 客户端
```java
public class HomeApplianceStore{
	private Refrigerator refrigerator; 
	private WashingMachine washingMachine;
	
	public HomeApplianceStore(Factory factory){
		this.refrigerator = factory.createRefrigerator();
		this.washingMachine = factory.createWashingMachine();
	}
	
	public void showProducts(){
		refrigerator.displayInfo(); 
		washingMachine.displayInfo(); 
		System.out.println();
	}
	
	public void playProducts(){
		refrigerator.cool(); 
		washingMachine.wash(); 
		System.out.println("────────────────────");
	}
	
	public static void main(String[] args){
		String brand = "Haier"; // 可改为 "Midea"
		
		ApplianceFactory factory = switch (brand.toLowerCase()) { 
			case "haier" -> new HaierFactory(); 
			case "midea" -> new MideaFactory(); 
			default -> throw new IllegalArgumentException("不支持的品牌"); 
		};
		
		HomeApplianceStore store = new HomeApplianceStore(factory);
		factory.showProducts();
		factory.playProducts();
		
		// 再演示切换品牌 
		System.out.println("客户换了美的系列：\n"); 
		HomeApplianceStore store2 = new HomeApplianceStore(new MideaFactory());
		store2.showProducts();
		store2.run();
	}
}
```

## 工厂
```java
public inferface Factory{
	Refrigertator createRefrigerator();
	WashingMachine createWashingMachine();
}
```

```java
public class HaierFactory implements Factory{
	@Override
	public Refrigertator createRefrigerator(){
		return new HaierRefrigertator();
	}
	@Override
	public WashingMachine createWashingMachine(){
		return new HaierWashingMachine();
	}
}
```

```java
public class MideaFactory implements Factory{
	@Override 
	public Refrigerator createRefrigerator() { 
		return new MideaRefrigerator();
	}

	@Override 
	public WashingMachine createWashingMachine() { 
		return new MideaWashingMachine(); 
	}
}
```
